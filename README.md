# My Knowledge Base

基于 VitePress 搭建的个人知识库/笔记网站，用于整理技术文档、学习笔记和工作心得。

👉 **在线访问**: [https://nbkelasi.github.io/docs-website/](https://nbkelasi.github.io/docs-website/)


## 特性

- 📚 Markdown 优先的文档编写
- ⚡ Vite 驱动的极速开发体验
- 🎨 默认主题 + 自定义样式扩展
- 🌐 自动生成侧边栏和导航
- 📱 响应式移动端适配
- 🔍 内置全文搜索（通过 Algolia 或本地搜索）
- 📦 GitHub Actions 自动部署
